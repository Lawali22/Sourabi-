## À propos du site

Site vitrine mono-page pour Sourabié Adama — conteur, auteur et metteur en scène burkinabè.

**Pages :**
- Accueil — Hero, présentation, aperçu des scènes
- Biographie — Parcours complet 2014 → 2025
- Prix & Distinctions — Réalisations et reconnaissances
- Médiathèque — Vidéos YouTube
- Sur Scène — Historique des invitations (2018–2025)
- Contact — Coordonnées de booking

## Déploiement sur Vercel

### Option 1 — Interface web (recommandé)

1. Créez un compte sur [vercel.com](https://vercel.com)
2. **Add New Project** → **Upload** → glissez le dossier
3. Cliquez **Deploy** → votre site est en ligne ✓

### Option 2 — Via CLI

```bash
# Installer Vercel CLI
npm install -g vercel

# Dans le dossier du projet
vercel

# Répondre aux questions :
# Set up and deploy? → Y
# Which scope? → votre compte
# Link to existing project? → N
# Project name? → sourabie
# Directory? → ./
# Override settings? → N
```

### Option 3 — Via GitHub (mise à jour automatique)

1. Créez un repo GitHub et poussez ce dossier
2. Sur Vercel → **Import Git Repository**
3. Chaque `git push` met à jour le site automatiquement
